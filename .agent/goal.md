This is a project to implement the ZOH runtime in C#, while providing feedbacks to the spec.

You need to follow the language spec and the implementation spec to develop the C# runtime from the ground up, and then maintain it in the future.

You should log your findings, questions, and feedbacks using the repomem mcp.

You should not rush, but carefully step through the implementation plan to build the runtime piece by piece, utilize git to track your progress, and make sure to test your implementation thoroughly with solid, foundamental tests on each components.
